## Bash Scripting

### Shebang

```bash
$ #!/bin/bash
```



### Comment

```bash
$ #!/bin/bash

# This is Comment
```



### Hello World

```bash
$ #!/bin/bash

# This is Comment
echo "Hello World"
```



### Read Script

```bash
$ cat hello.sh
```



### Set Permission (CHMOD)

```bash
$ chmod 755 hello.sh
```



### Execute Script

```bash
$ ./hello.sh
```
